DVLD - Driving License Management
Overview
The Driving License Management (DVLD) system is a comprehensive desktop application designed to streamline operations for driving authorities. It handles the management of drivers, examinations, and vehicle registrations while tracking progress and automating routine tasks. The system is built to ensure efficiency, accuracy, and scalability.

Features
1. Driver Management
Add, update, and maintain driver profiles.
Track driver progress through exams and licensing stages.
2. People Management
Manage records of examiners, administrators, and other personnel.
Assign roles and responsibilities to staff efficiently.
3. Exam Management
Schedule, manage, and evaluate driving theory and practical exams.
Record exam results and update driver progress automatically.
4. License Issuance
Issue, renew, or revoke driver licenses.
Generate digital and printable license documents.
5. Vehicle Registration
Maintain vehicle records, including ownership, inspections, and registrations.
Track changes in ownership and regulatory compliance.
6. Reporting and Progress Tracking
Generate detailed reports on driver progress, exam results, and staff performance.
Analyze licensing data for decision-making and operational improvements.
Technologies Used
Programming Language: C#
Framework: .NET Framework
Database: SQL Server, T-SQL
How to Run the Project
1- Clone this repository to your local system.
2- Install and configure SQL Server.
3- Set up the database using the provided schema files.
4- Adjust the connection string in the configuration file.
5- Open the project in Visual Studio, build the solution, and run the application.
